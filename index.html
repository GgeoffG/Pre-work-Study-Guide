HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Opening and closing tags are enclosed within angled brackets.

    • There are 4 parts of an html element

    • The opening tag: Consists of the name of the element (such as <p>,<h>,<li>) wrapped in angle brackets. Idicates the beggining of an element.
    • The closing tag: Consists of the same structure as the opening tag, with a Forward Slash before the element name. Indicates where the element ends. "
    • The content: What is inside the element much like this text is indie the <li> element
    • The element: The sum total of the opening/closing tags and the content makes the element.

    Elements can also have attributes

    • Attributes are extra information about an element that won't appear in the frontend content. An atrribute might look something like this:

      • <p class="editor-note">

      This attributes Class as the attribute name and editor-note as the attributes value. Suggesting that the "p" paragraph is labeled under the name "editor note".

    • There are three things an attribute that set a value always has

      • A space between it and the element name or previous attribute
      • The Attribute name followed by an equal sign
      • The attribute value wrapped by quotation marks.

    • Nesting Elements

      • Elements can be placed inside other elements
      • This is known as nesting
      • <strong> is nested within the prievous <li> element

    • Void Elements

      • Some elements have no content and are called "Void Elements" Elements such as <img> do not wrap content to affect it and are instead used to embed and image in the htmal page where it appears
